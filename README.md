# MeteoApp-project

My student project. A simple, clear and responsive weather web app. It's available in Polish language.

## Project Setup

```sh
npm install
```


### Compile and Hot-Reload for Development

```sh
npm run dev
```


### Compile and Minify for Production

```sh
npm run build
```
